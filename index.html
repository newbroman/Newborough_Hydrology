About the study

Newborough Warren, Isle of Anglesey

Newborough Warren is one of Wales's most significant coastal sand dune systems and a designated Special Area of Conservation (SAC) under the EU Habitats Directive. Its wet dune slacks — low-lying interdune hollows that flood seasonally — support a range of priority habitats, including H2180 Wooded Dunes, and provide the hydrological conditions on which rare dune slack communities depend. View site map →

97 dipwells monitored
21 years 2005–2026
890 mm mean annual rainfall
5 hydrogeological clusters

The study uses a State Space Model (SSM) fitted to monthly groundwater records across a 97-well dipwell network. Hierarchical clustering identifies five hydrogeological behavioural clusters (C1–C5), including C4, a Corsican pine plantation in which canopy interception suppresses recharge. The principal finding is that plantation clearance does not raise winter water table maxima: post-felling, atmospheric draw (β₂) deepens summer minima, which is the dominant control on winter flooding probability. This counterintuitive result is supported by a Before-After-Control-Impact (BACI) analysis of two clearance events and by independent spatial validation of a ridge-derived water balance residual.

The monitoring network was designed to fill the calibration gap identified by Bristow (2003), whose MODFLOW model failed due to dry cells and a sparse observation record during an anomalously dry calibration decade. The 21-year dipwell record reported here provides the basis for a fully empirical characterisation of groundwater dynamics at the site.

Data: Dipwell records © M. Hollingham. Meteorological inputs from RAF Valley (~16 km north-west). The study was conducted in support of conservation management by Natural Resources Wales (NRW).

Citation: Hollingham, M. (2026) Hydrogeological Dynamics, Behavioural Clustering and Management Intervention Analysis at Newborough Warren Coastal Sand Dune Aquifer, Wales. Journal of Hydrology: Regional Studies (submitted).

Running the pipeline

Reproducing the analysis

The full analytical pipeline runs 28 steps across eleven phases, coordinated by run_analysis.py. All file paths are managed centrally through src/utils/paths.py; no hardcoded paths appear in individual scripts. Python 3.10 or later is required (3.12 tested).

# Clone the repository
git clone https://github.com/newbroman/Newborough_Hydrology.git
cd Newborough_Hydrology

# Create a virtual environment and install dependencies
python -m venv .venv
source .venv/bin/activate    # Windows: .venv\Scripts\activate
pip install -r requirements.txt

# Launch the interactive pipeline menu
python run_analysis.py

# Or run non-interactively
python run_analysis.py --full      # run all 28 steps
python run_analysis.py --from 14  # resume from step 14
python run_analysis.py --viewer   # build scenario viewer only
Note: KML support requires Fiona with the LIBKML driver enabled. Stream network skeletonisation (script 20) requires scikit-image. The outputs/ directory should be excluded from version control. See readme.md in the repository for a full description of each script's inputs, outputs, and ordering constraints.
